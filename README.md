# Agent Library

个人 AI Agent 技能库，包含 skills、rules、prompts 和 configs。

## 目录结构

- `skills/` - Agent 技能（SKILL.md + 相关脚本/引用文件）
- `rules/` - 规则文件
- `prompts/` - Prompt 模板
- `configs/` - 配置文件
